## Step 3 — 全部領取 分批领 → 剩 meta任务「完成每日任務8次」单领
帧 2363x1331 · 多次 `全部领取_黄`→`获得奖励`→`点击继续字样` 后：
- **`全部領取` 变灰**（完成的常规任务奖励已领完，分 2~3 批 reward 弹窗）
- 剩 `领取_黄`(0.760,0.927) = 「**完成每日任務8次以上 8/8**」meta 奖励（领满8个日常后解锁，需单领）
- 其余任务都是 `立即前往`(go)=**未完成**：制約解除決戰(活动)/收集裝備設計圖/挑戰任務(Hard/Normal campaign)
  → skill **不自动做这些**（事件/挑战/campaign），只领已完成的

★★ **daily-mission 领奖逻辑**：
  1. 循环 `全部领取_黄` → reward弹窗(reveal→点击继续字样 dismiss) 直到 `全部领取` 变灰
  2. 再扫剩余单个 `领取_黄`（如 meta「完成每日任務X次」）逐个领
  3. 切各 tab（每天/每週/成就/挑戰任務）分别重复 全部领取
  4. 无 黄(全部领取_黄/领取_黄) = done
  ⚠️ `立即前往` 按钮的任务**不碰**（未完成的事件/挑战/campaign）
- 新 cls：`全部领取_黄` / `领取_黄` / `立即前往`(go按钮)（区别于 mail/arena 的领取态）
- ★ 顺序：daily-mission 必须**最后**跑（其他日常完成才解锁这里的任务奖励）
→ 点 `领取_黄`(0.760,0.927) 领 meta 奖励
