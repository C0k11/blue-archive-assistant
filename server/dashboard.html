UI 干扰说明 (训练 augmentation, 不是吸附框):
• Lv/Star/Weapon/Heart = 随机位置随机大小贴在 slot 内, 让 model 学会"忽略 UI 装饰看头像本体"
• 这些位置不能手动微调, 概率越高越多张合成图会带上对应干扰
• Border ablation = 随机遮掉头像 4 边一小条, 学头像中心不学边框
